 
# 📊 Mutual Funds Analysis

## 📌 Overview

This project performs an in-depth analysis of Indian Mutual Funds using Python. It cleans and analyzes the dataset to uncover insights about fund performance, risk levels, categories, Asset Management Companies (AMCs), and historical returns.

The project leverages Exploratory Data Analysis (EDA) techniques to help investors compare mutual funds and make data-driven investment decisions.

---

## 🚀 Features

- Data Cleaning & Preprocessing
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Category-wise Performance Analysis
- Risk-wise Return Comparison
- AMC-wise Fund Analysis
- Fund Rating Analysis
- Top Performing Mutual Funds
- Pivot Table Analysis
- Data Visualization using Matplotlib & Seaborn
